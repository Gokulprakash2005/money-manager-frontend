# Money Manager Frontend

A React-based web application for managing personal finances with income and expense tracking.

## Features

- Add income and expenses with categories and divisions
- Filter transactions by type, category, division, and date range
- View weekly, monthly, and yearly summaries
- Edit transactions within 12 hours
- Responsive design with Tailwind CSS

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file with:
```
REACT_APP_API_URL=http://localhost:5000/api
```

3. Start development server:
```bash
npm start
```

## Build for Production

```bash
npm run build
```

## Technologies Used

- React 18
- Tailwind CSS
- Axios for API calls
- Lucide React for icons
- React Router for navigation
